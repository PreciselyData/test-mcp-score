# Precisely MCP Servers

This repository contains the active Precisely Model Context Protocol (MCP) server implementation for cloud APIs.

## Repository Structure

```
precisely-mcp-servers/
└── dis-locate-apis-v2/
```

## Quick Start

1. Open the active server project:

```bash
cd dis-locate-apis-v2
```

2. Follow the setup and usage guide in:

`./dis-locate-apis-v2/readme.md`

## Authentication

The server requires Precisely API credentials:

- API Key
- API Secret

Configuration details are documented in `dis-locate-apis-v2/readme.md`.

- API documentation: https://developer.cloud.precisely.com/apis
